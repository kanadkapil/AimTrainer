# Aim Training Simulator

A Python-based aim trainer using Pygame and AI adaptation.

## Features

- **Multiple Game Modes**: Flick, Tracking, Reflex.
- **AI Adaptation**: Adjusts difficulty based on player performance using Machine Learning Basic Heuristics.
- **Analytics**: Visualizes aim path, reaction time, and accuracy.

## Installation

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the game:
   ```bash
   python main.py
   ```
